Сагайдак Кирилл разработал программу по теме "Текстовый редактор". Основные функции, которые он реализовал:
1) Подсчёт количества слов в тексте - вызывается при вводе в консоль цифры 1
2) Поиск самого длинного слова - вызывается при вводе в консоль цифры 2
3) Определение частоты использования символов - вызывается при вводе в консоль цифры 3
4) Удаление лишних пробелов - вызывается при вводе в консоль цифры 4
5) Преобразование регистра (верхний/нижний) - вызывается при вводе в консоль цифры 5
6) Автоматическое выравнивание текста по ширине - вызывается при вводе в консоль цифры 6
При запуске программы предлагается ввести текст для дальнейшего его анализа. Если после ввода текста ввести цифру 7, то можно написать новый текст. Если ввести 0 - программа завершится.
